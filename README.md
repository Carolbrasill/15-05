## 💻 Sobre o Projeto

Este projeto foi criado para documentar e organizar os códigos, algoritmos e lógicas desenvolvidos durante os meus treinamentos para a **WorldSkills na ocupação #23 Robótica Móvel**. O objetivo principal é desenvolver soluções ágeis e precisas para os desafios de navegação, mapeamento e manipulação de objetos exigidos nas provas oficiais.

---

## ✨ Funcionalidades e Roadmap (Metas Futuras)

Aqui está o planejamento do projeto, dividindo o que já foi iniciado na programação/simulação e o que será implementado no hardware físico.

### 💻 Já Iniciado (Programação e Simulação)
- [x] **Lógica Base de Controle:** Primeiros passos na lógica de tomada de decisão usando Java e Kotlin.
- [x] **Simulação de Circuitos:** Testes iniciais de componentes e lógica de sensores dentro do Tinkercad.

### 🤖 Metas Futuras (Implementação no Robô Físico)
- [ ] **Movimentação e Odometria:** Ajustar o controle dos motores para deslocamento real em milímetros e curvas exatas.
- [ ] **Integração de Sensores:** Conectar e tratar os dados físicos dos sensores (ultrassônico, infravermelho, bumpers, etc.).
- [ ] **Mapeamento e Navegação:** Desenvolver algoritmos reais para o robô desviar de obstáculos na arena.
- [ ] **Manipulação Mecânica:** Programar garras e atuadores para coletar e posicionar os elementos da prova.

---

## 🛠️ Tecnologias Utilizadas

O desenvolvimento do robô envolve o seguinte conjunto de ferramentas técnicas:

- **Linguagem Principal:** Java com uma base em Kotlin (fornecida pelos treinadores) - Linguagem base utilizada para escrever a lógica de controle e tomada de decisão do robô.
- **Plataforma:** Tinkercad e Arduino - Ambiente de simulação e hardware principal para comunicação e gerenciamento dos componentes do robô.
- **Git & GitHub** - Para versionamento de código e histórico detalhado das alterações.

## 🚀 Como Executar o Projeto

> ⚠️ **Nota:** Atualmente, o projeto está em fase de simulação e desenvolvimento lógico, preparando a base para o hardware real.

### 🛠️ Pré-requisitos
Para abrir e visualizar este projeto, você precisará de:
- Uma conta no [Tinkercad](https://www.tinkercad.com/) (para visualizar os circuitos simulados).
- Uma IDE de desenvolvimento instalada (como IntelliJ IDEA para Java/Kotlin ou a Arduino IDE).

### 🎲 Como Testar
Como o projeto está dividido entre código e simulação, você pode:

1. **Ver os Códigos:** Navegue pelas pastas deste repositório para analisar a lógica desenvolvida em Java e Kotlin.
2. **Simulação:** Copie a lógica de circuitos desenvolvida para testar os componentes no ambiente do Tinkercad.
